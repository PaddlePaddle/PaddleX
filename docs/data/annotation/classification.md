# 图像分类数据标注

图像分类标注是一项最基础，最简单的标注任务，图像分类无需标注工具，只需将属于同一类的图片放在同一个文件夹下即可，例如下所示目录结构，
```
MyDataset/ # 图像分类数据集根目录
|--dog/ # 当前文件夹所有图片属于dog类别
|  |--d1.jpg
|  |--d2.jpg
|  |--...
|  |--...
|
|--...
|
|--snake/ # 当前文件夹所有图片属于snake类别
|  |--s1.jpg
|  |--s2.jpg
|  |--...
|  |--...
```
